1. FASE PREPARATORIA
1.1 Registro Obligatorio Inicial (Ambos: OIV y PSE)
Basado en Instrucción General N°1 de la ANCI
Requisitos Previos:
•	✅ Entidad calificada como PSE según Art. 4 Ley 21.663
•	✅ Designación formal del encargado de ciberseguridad
•	✅ Clave Única del encargado + 2FA activado
Proceso de Registro:
1.	Acceso a Portal: https://portal.anci.gob.cl
2.	Documentación Requerida: 
o	Documento de designación firmado con FEA por representante legal
o	Certificado de vigencia de la sociedad
o	Certificado de vigencia de poderes
3.	Información a Registrar: 
o	Datos de la institución (razón social, RUT, sector)
o	Datos del encargado de ciberseguridad
o	Correo electrónico institucional oficial

2. TAXONOMÍA DE INCIDENTES SEGÚN RESOLUCIÓN EXENTA N°7
Clasificación por Áreas de Impacto:
A. Impacto en el uso legítimo de recursos
•	Uso no autorizado de redes y sistemas informáticos
•	Actividades de phishing o fraude en infraestructura propia
•	Actividades de phishing o fraude relacionadas con la institución
•	Ejecución no autorizada de código
B. Impacto en la confidencialidad de la información
•	Exfiltración y/o exposición de datos
•	Exfiltración y/o exposición de configuraciones
•	Exfiltración y/o exposición de código fuente
C. Impacto en la disponibilidad de un servicio esencial
•	Indisponibilidad y/o denegación de servicio
•	Degradación de servicio
D. Impacto en la integridad de la información
•	Modificación no autorizada de datos
•	Manipulación no autorizada de configuración
________________________________________
3. CRITERIOS DE "EFECTO SIGNIFICATIVO" (DS N°295, Art. 3)
Un incidente debe reportarse si es capaz de:
•	✅ Interrumpir la continuidad de un servicio esencial
•	✅ Afectar la integridad física o la salud de las personas
•	✅ Comprometer la confidencialidad de archivos con datos personales
•	✅ Implicar acceso no autorizado a redes o sistemas informáticos
•	✅ Involucrar sistemas que contengan datos personales

5. DIFERENCIAS CLAVE POR TIPO DE ENTIDAD
5.1 Prestadores de Servicios Esenciales (PSE)
Plazos:
•	⏰ Alerta Temprana: 3 horas
•	⏰ Segundo Reporte: 72 horas
•	⏰ Informe Final: 15 días
Características:
•	✅ No requiere Plan de Acción específico
•	✅ Multas menores (hasta 20,000 UTM máximo)
•	✅ Proceso de reporte estándar
5.2 Operadores de Importancia Vital (OIV)
Plazos:
•	⏰ Alerta Temprana: 3 horas
•	⏰ Segundo Reporte: 
o	24 horas (si servicio esencial afectado)
o	72 horas (otros casos)
•	⏰ Plan de Acción: 7 días
•	⏰ Informe Final: 15 días
Características:
•	✅ Requiere Plan de Acción detallado
•	⚠️ Multas mayores (hasta 40,000 UTM máximo)
•	✅ Obligaciones adicionales (SGSI, auditorías, certificaciones)
•	✅ Delegado de ciberseguridad obligatorio

6. LÓGICA DE IMPLEMENTACIÓN TÉCNICA
6.1 Sistema de Gestión de Incidentes Interno
class IncidenteANCI:
    def __init__(self, tipo_entidad):
        self.tipo_entidad = tipo_entidad  # "OIV" o "PSE"
        self.estado = "DETECTADO"
        self.timeline = []
        self.reportes_enviados = []
        
    def determinar_plazos(self, servicio_afectado=False):
        if self.tipo_entidad == "OIV" and servicio_afectado:
            return {
                "alerta_temprana": 3,  # horas
                "segundo_reporte": 24,  # horas
                "plan_accion": 7 * 24,  # horas (7 días)
                "informe_final": 15 * 24  # horas (15 días)
            }
        else:
            return {
                "alerta_temprana": 3,  # horas
                "segundo_reporte": 72,  # horas
                "plan_accion": None,  # No aplica para PSE
                "informe_final": 15 * 24  # horas (15 días)
            }
    
    def evaluar_efecto_significativo(self, incidente_data):
        criterios = [
            incidente_data.get("interrupcion_servicio", False),
            incidente_data.get("afecta_salud_personas", False),
            incidente_data.get("datos_personales_comprometidos", False),
            incidente_data.get("acceso_no_autorizado", False)
        ]
        return any(criterios)

6.3 Sistema de Alertas y Recordatorios
def configurar_alertas(incidente):
    plazos = incidente.determinar_plazos()
    
    # Alertas automáticas
    alertas = [
        {"tipo": "alerta_temprana", "tiempo": 1, "unidad": "horas"},
        {"tipo": "segundo_reporte", "tiempo": plazos["segundo_reporte"] - 6, "unidad": "horas"},
        {"tipo": "informe_final", "tiempo": plazos["informe_final"] - 48, "unidad": "horas"}
    ]
    
    if incidente.tipo_entidad == "OIV" and plazos["plan_accion"]:
        alertas.append({
            "tipo": "plan_accion", 
            "tiempo": plazos["plan_accion"] - 24, 
            "unidad": "horas"
        })
    
    return alertas

8. CONSIDERACIONES LEGALES IMPORTANTES
Sanciones por Incumplimiento:
•	PSE: Hasta 20,000 UTM por infracciones gravísimas
•	OIV: Hasta 40,000 UTM por infracciones gravísimas
Infracciones Típicas:
•	No reportar en plazo (3 horas): GRAVE
•	Información falsa o incompleta: GRAVÍSIMA
•	No inscribirse en plataforma ANCI: LEVE
Protección Legal:
•	Principio non bis in idem: No doble sanción por mismo hecho
•	Derecho a recursos administrativos y reclamación judicial
•	Confidencialidad de información sensible reportada
